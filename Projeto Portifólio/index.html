Gerador de Senhas

Este projeto é uma aplicação web desenvolvida com HTML, CSS e JavaScript, cujo principal objetivo é fornecer uma ferramenta prática, segura e acessível para a geração de senhas personalizadas. Ele foi pensado tanto para fins educacionais (para quem está aprendendo desenvolvimento web) quanto para uso real por usuários que desejam criar senhas fortes para contas e serviços online.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em uma divisão principal (.container-gerador-senha) que apresenta os campos para gerar as senhas. Cada campo é destacado em sua própria div, mostrando um container alternativas para que o usuário possa gerar a sua senha aleatória de acordo com o seu desejo. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site.

Gerador de Pokemons

A Pokédex é uma aplicação web que permite aos usuários buscar e visualizar informações detalhadas sobre os Pokémons, simulando a famosa enciclopédia digital do universo Pokémon. Este projeto foi desenvolvido inteiramente com tecnologias de front-end: HTML, CSS e JavaScript, e utiliza a PokéAPI para obter os dados em tempo real.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em uma tag de principal "main" (.container) que uma imagem do primeiro Pokemon da Pokedex da API utilizada . Cada campo é destacado em tags de um título (h1), o nome e o número do Pokemon, um para inserir o nome ou número do Pokemon que deseja procurar (input) e dois botões para buscar (button). O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos.
